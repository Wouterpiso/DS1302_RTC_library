Arduino C++ library for the {\bfseries{DS1302 Real Time Clock (RTC)}} module.

This library provides a simple interface to read and write date/\+time from the DS1302 chip using a 3-\/wire communication protocol.

\DoxyHorRuler{0}
\hypertarget{index_autotoc_md2}{}\doxysection{\texorpdfstring{✨ Features}{✨ Features}}\label{index_autotoc_md2}

\begin{DoxyItemize}
\item Read and write date \& time
\item Simple 3-\/wire interface (CE, I/\+O, SCLK)
\item Compatible with Arduino, ESP8266, ESP32
\item Low-\/level register access (if exposed in implementation)
\item Optional RAM read/\+write support
\item Alarm functionality is software-\/based.
\end{DoxyItemize}
